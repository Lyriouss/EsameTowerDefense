# EsameTowerDefense



Extra features added that were not requested to exam

* Turret Aiming: Turrets will target enemies inside it's range and primarily target enemy closest to player base.
* Extra Upgrade Stat: Machine Gun and Area Turrets upgrade also increases damage by a certain amount on top of Machine Gun Turret's rate of fire and Area Turret's bullet area.
* Added Upgrade Limit: Turrets can only be upgraded a total of 5 times until they reach MAX level.
* Sniper Turret: Added an extra Turret that generates bullets that hits all enemies in it's path (does not destroy upon hitting an enemy). Each upgrade increases it's range and damage. Placement/upgrade cost is equivalent to €20.
* Enemy Spawn Positions: Spawns enemies randomly across three different lanes.
* Enemy Health Increase: After spawning a certain number of enemies, raises by one the health multiplier at which enemies spawn. Example: Starting Mult = 1, Increase After = 30, Enemy Health = 10; after 30 spawns Starting Mult = 2 and enemies now spawn with 10 \* 2 = 20 Health ecc...
* Enemy Stealing: When enemy reaches player base, steals from player the amount of money that would've been rewarded upon killing (because why not C:).
* Added UI Elements: -

 	-Extra button added for Sniper Turret Placement

 	-Kill counter added

 	-Text that shows what Turret placement option is selected

 	-Unselect Button that removes current Turret placement option so placement option is not always selected to a 	specific Turret

 	-Delete Button that when selected makes position under Turret red and if Turret is clicked, destroys that 	Turret.
